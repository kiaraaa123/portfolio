# Ki'ara Colbert Site Rebrand Starter

This is a lightweight static portfolio starter built with HTML, CSS, and a tiny bit of JavaScript.

## Files
- `index.html` - page content and structure
- `styles.css` - visual design and responsive layout
- `script.js` - mobile menu behavior

## How to use
1. Open `index.html` in your browser to preview.
2. Replace text with your final copy.
3. Add your actual links, resume link, and any images you want.
4. Upload to your existing hosting/GitHub Pages setup.
